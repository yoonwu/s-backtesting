# RSI 다이버전스(변곡) 전략 전수 그리드서치

리포 루트의 MT5 CSV(US100·XAUUSD, 2024-07~2026-06)로 336개 조합을 전수 백테스트한다.
이전 세션에서 주식/지수 일봉(PyPI 번들 데이터)으로 돌렸던 동일 그리드를 실거래 대상 데이터로 재검증한 것.

## 실행

```bash
pip install numpy pandas
python3 divergence/gridsearch.py
```

약 10초 내 완료. 결과는 `divergence/results/`에 저장:

- `grid_results_by_dataset.csv` — 336조합 × 10데이터셋 (3,360행) 상세
- `grid_results_agg.csv` — 조합(336) 단위 전 데이터셋 합산
- `top10.md` — 요약 리포트 (핵심 발견·능선 검증·다음 단계)

## 전략 정의 (gridsearch.py 도크스트링과 동일)

1. **스윙 피벗**: 좌우 k봉(3/5) 극값. k봉 뒤 확정(룩어헤드 방지).
2. **다이버전스**: 연속 피벗에서 가격 저점↓ + RSI(9/14) 저점↑ = 롱(숏은 대칭).
3. **자리 필터**(컨플루언스 AND, 7가지 부분집합):
   - `SR` 과거 피벗 레벨 ±0.5×ATR14 / `BB` 볼린저(20,2σ) 밴드 이탈 / `FIB` 최근 120봉 레인지 38.2/50/61.8% 되돌림 ±0.5×ATR14
4. **트리거**(확정봉부터 10봉 내): `캔들`(장악형) / `MACD`(12,26,9 크로스) / `캔들+MACD`
5. **체결**: 트리거 다음 봉 시가 진입, 손절=트리거봉 극값, 익절 1.5R/2.5R.
   보수적 처리 — SL·TP 동시 도달 시 SL, 갭 손절은 시가 체결, 최대 300봉 보유, 동시 1포지션.

## 핵심 결론 (2026-07-02)

- 합산 상위권 골격: **롱 · FIB 포함 자리 · 캔들+MACD · 2.5R · k=5** (EA필터 통과 66/336)
- 능선 최우수: **XAU H2 롱 SR+BB 캔들 (8/8칸 양수, PF 2.1~4.4)** — 단 연 11회로 표본 얇음
- 표본·능선 균형 최우수: **XAU M30 롱 BB 캔들+MACD k5 (73거래, PF 1.84)** — 비용 검증 후보 1순위
- 숏은 전반 열세이나 **US100 H1 숏 SR+BB k5**가 상승장 2년 내내 양수(헤지 후보)
