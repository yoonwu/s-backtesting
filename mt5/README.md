# MT5 자동매매 시스템 — DoubleBB_EA

LEVERAGE LAB 더블비 백테스트를 MT5에서 그대로 돌리는 Expert Advisor.
**하나의 EA(`DoubleBB_EA.mq5`)**를 각 차트에 다른 입력값·매직넘버로 붙여서 5개 전략을 동시에 운용한다.

---

## ⚠️ 시작 전 필수

1. **헤징(Hedging) 모드 계좌** 사용. 네팅 계좌는 롱/숏 상계로 전략이 서로 잡아먹음.
2. **데모/전략테스터에서 먼저 검증.** 아래 입력값으로 Strategy Tester를 돌려 백테스트 수치(수익률·MDD·거래수)와 비슷하게 나오는지 확인한 뒤에만 실거래.
3. **거래비용 확인.** 골드 M5 숏(④)·골드 M5 롱(⑦)은 스프레드에 취약. MT5 틱으로 실제 왕복 스프레드 보고 비중 결정.
4. 터미널 우상단 **알고리즘 트레이딩 버튼 ON**, EA 속성에서 자동매매 허용.

## 설치
1. MetaEditor → `MQL5/Experts/` 폴더에 `DoubleBB_EA.mq5` 복사 → **컴파일(F7)**.
2. 각 전략별 차트를 열고 (종목+타임프레임 정확히), EA를 드래그.
3. 입력 탭에서 아래 표대로 값 입력. **매직넘버는 전략마다 반드시 고유값.**

---

## 확정 5전략 입력값

> 공통: `InpLotPerEntry`=차수당 랏(자금에 맞게), `InpExpiryBars`=50, `InpMaxSpreadPts`=0(또는 브로커 스프레드 기준 상한).
> 분할 차수당 랏 × 분할수 = 한 셋업 최대 노출. 예) 등분4 × 0.01랏 = 최대 0.04랏.
>
> ⚠️ **2026-06-29 Hantec 종목 변경**: 지수 소형심볼 `.b`(US100.b 등)가 **청산전용(Close Only)**으로 전환됨 → 신규진입 불가.
> 나스닥은 **`US100.x`**(계약단위 10, 최소랏 0.01)로 거래. `US100.x` 0.01랏 = 옛 `US100.b` 0.1랏과 동일 노출.
> 커미션 6 USD/랏. 골드(XAUUSD)는 변경 대상 아님(거래 전 명세서에서 "거래: 전체 가능" 재확인 권장).

### 비중 가이드
- **큰 비중**: ⑤⑥ (비용 면역·고손익비·느린 스윙)
- **중간**: ①⑦ (M5지만 손익비 양호)
- **작게·타이트 스프레드만**: ④ (승률 낮아 비용 취약). 단 유일한 숏이라 헤지 역할 — 0으로 빼진 말 것.

## 운용 메모
- **2티어 구조**: 스윙(⑤⑥, 묻어두기) + 인트라데이(①④⑦, 관리형).
- **골드 묶음**: ④숏 + ⑥롱 + ⑦롱. ⑦과 ④는 M5 롱/숏이라 MA필터가 방향을 갈라 자연 헤지.
- 신호 빈도: ⑥은 월 1~2회(드물어도 정상), M5들은 자주.
- 모든 전략 **동시 1포지션** 원칙(셋업당 한 번에 한 포지션만).

## ⚠️ EA 한계 / 검증 포인트
- 이 EA는 백테스트 로직의 **충실한 이식이지만 100% 동일 보장은 아님.** 라이브에서는 지정가 체결가·슬리피지·갭·스프레드가 백테스트와 달라짐.
- **반드시 Strategy Tester(틱 데이터)로 동일 입력 백테스트** 후 수치 대조 → 큰 괴리 없으면 데모 1~2주 → 소액 실거래 순으로.
- 분할 지정가가 이미 지나간 가격이면 거부될 수 있음(로그 확인).
- 주말/롤오버/뉴스 갭 구간은 별도 리스크. 필요시 거래시간 필터·뉴스 회피 추가.
