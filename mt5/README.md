# MT5 자동매매 시스템 — DoubleBB_EA

LEVERAGE LAB 더블비 백테스트를 MT5에서 그대로 돌리는 Expert Advisor.
**하나의 EA(`DoubleBB_EA.mq5`)**를 각 차트에 다른 입력값·매직넘버로 붙여서 5개 전략을 동시에 운용한다.

---

## ⚠️ 시작 전 필수

1. **헤징(Hedging) 모드 계좌** 사용. 네팅 계좌는 롱/숏 상계로 전략이 서로 잡아먹음.
2. **데모/전략테스터에서 먼저 검증.** 아래 입력값으로 Strategy Tester를 돌려 백테스트 수치(수익률·MDD·거래수)와 비슷하게 나오는지 확인한 뒤에만 실거래.
3. **거래비용 확인.** 골드 M5 숏(④)·골드 M5 롱(⑦)은 스프레드에 취약. MT5 틱으로 실제 왕복 스프레드 보고 비중 결정.
4. 터미널 우상단 **알고리즘 트레이딩 버튼 ON**, EA 속성에서 자동매매 허용.

## 설치
1. MetaEditor → `MQL5/Experts/` 폴더에 `DoubleBB_EA.mq5` 복사 → **컴파일(F7)**.
2. 각 전략별 차트를 열고 (종목+타임프레임 정확히), EA를 드래그.
3. 입력 탭에서 아래 표대로 값 입력. **매직넘버는 전략마다 반드시 고유값.**

---

## 확정 5전략 입력값

> 공통: `InpLotPerEntry`=차수당 랏(자금에 맞게), `InpExpiryBars`=50, `InpMaxSpreadPts`=0(또는 브로커 스프레드 기준 상한).
> 분할 차수당 랏 × 분할수 = 한 셋업 최대 노출. 예) 등분4 × 0.01랏 = 최대 0.04랏.
>
> ⚠️ **2026-06-29 Hantec 종목 변경**: 지수 소형심볼 `.b`(US100.b 등)가 **청산전용(Close Only)**으로 전환됨 → 신규진입 불가.

> 커미션 6 USD/랏. 골드(XAUUSD)는 변경 대상 아님(거래 전 명세서에서 "거래: 전체 가능" 재확인 권장).

### 비중 가이드
- **큰 비중**: ⑤⑥ (비용 면역·고손익비·느린 스윙)
- **중간**: ①⑦ (M5지만 손익비 양호)
- **작게·타이트 스프레드만**: ④ (승률 낮아 비용 취약). 단 유일한 숏이라 헤지 역할 — 0으로 빼진 말 것.

## 운용 메모
- **2티어 구조**: 스윙(⑤⑥, 묻어두기) + 인트라데이(①④⑦, 관리형).
- **골드 묶음**: ④숏 + ⑥롱 + ⑦롱. ⑦과 ④는 M5 롱/숏이라 MA필터가 방향을 갈라 자연 헤지.
- 신호 빈도: ⑥은 월 1~2회(드물어도 정상), M5들은 자주.
- 모든 전략 **동시 1포지션** 원칙(셋업당 한 번에 한 포지션만).

## ⚠️ EA 한계 / 검증 포인트
- 이 EA는 백테스트 로직의 **충실한 이식이지만 100% 동일 보장은 아님.** 라이브에서는 지정가 체결가·슬리피지·갭·스프레드가 백테스트와 달라짐.
- **반드시 Strategy Tester(틱 데이터)로 동일 입력 백테스트** 후 수치 대조 → 큰 괴리 없으면 데모 1~2주 → 소액 실거래 순으로.
- 분할 지정가가 이미 지나간 가격이면 거부될 수 있음(로그 확인).
- 주말/롤오버/뉴스 갭 구간은 별도 리스크. 필요시 거래시간 필터·뉴스 회피 추가.

# 🆕 MA_Pullback_EA — 120선 눌림목 (더블비와 별개 EA)

`MA_Pullback_EA.mq5` = 웹 백테스터 **120선 눌림목** 탭 로직의 MT5 이식. **더블비와 완전히 별개 전략군**(돌파 아님, 눌림 매수).
설치: MetaEditor → `MQL5/Experts/`에 복사 → **F7 컴파일** → 차트에 드래그 → 입력값 설정.

## 로직
- **무장**: 종가가 이평보다 이격% 이상 위(롱)/아래(숏)로 벌어짐
- **진입**: 되돌림으로 이평선 **첫 터치**(롱=저가≤MA, 숏=고가≥MA) → 터치가(=MA) 기준 ∓KR, ∓2KR… N차 분할
- **KR** = 최근 N시간 최대봉(고가−저가). 비농·FOMC급 큰봉(평균봉×배수↑)은 큰봉지속(기본24h) 동안 KR 유지
- **손절** = 최심 매수가 ∓ 손절×KR (**가까운 실손절** 권장). **익절** = 평단 ± 익절×KR
- 브로커측 SL/TP 자동 부착(EA/PC 꺼져도 보호)

## ★ 확정 후보(전수조사 통과) — 2026-06-30
> 필터: 거래≥30·손절 실발동≥5회·PF≥0.5·전후반 둘 다 +. 공통: InpTRHours=12, InpNewsHours=24, InpNewsMult=3, InpTouchWaitH=168, InpExpiryBars=240, 익절 평단기준.

| 입력 | ★단기 US100 M30 | ★스윙 US100 H4 | ★골드스윙 XAU H2 | ★골드단기 XAU M10 숏 |

| **차트 종목** | US100.x | US100.x | XAUUSD | XAUUSD |
| **차트 TF** | **M30** | **H4** | **H2** | **M10** |
| InpDir | LONG | LONG | LONG | **SHORT** |
| **InpMAPeriod** | **120** | **200** | **180** | **200** |
| **InpGapPct** | **1.0** | **1.0** | **1.5** | **1.0** |
| **InpSplitCount** | **4** | **3** | **3** | **1**(단일) |
| **InpSL_x** | **1.0** | **1.0** | **0.5** | **1.5** |
| **InpTP_x** | 3.0 | **2.0** | **1.5** | **3.0** |
| InpLot1~N | 자금맞게 | 자금맞게 | 자금맞게 | InpLot1만 사용 |

- **★확정 4종 = 단기 US100 M30 · 스윙 US100 H4 · 골드스윙 XAU H2 · 골드단기 XAU M10 숏**. US100 2개 + 골드 롱/숏 = **종목·방향 분산**.
- **H4** *PF2.69는 이격1.0% 한정(0손절), 이격0.5%면 PF1.22 → 진짜 1.2~2.7. 20거래·상승장만이라 비중 작게.
- **골드 H2** = 랜덤3개월×30 25/27(최악-0.1%)·최근6개월+9.2%로 골드 눌림에도 버팀. ⚠️ **이격 1.5 고정**(1.25%↓면 폭망, 골드 전용 선별임계값).
- **골드 M10 숏** = **단일(N=1)이 핵심**(N1 PF2.00 > N2 1.22 > N3 0.84, 숏은 위로 물타기가 독). MA200이 곧 약세필터 → 별도 추세필터 불필요. 1년차 골드 +42% 폭등에도 +10.4%(MDD-1.8%)·2년차 +29.4%로 **올웨더**. 골드 H2 롱과 방향 반대 = **자연 헤지**. 단일이라 **InpSplitCount=1**, InpLot1만 사용.
- 매직넘버 **88000x**대 사용(더블비 99000x와 구분).
- 단기 대안: US100 M15(MA120·이격0.5%·3분할·SL0.5·TP3, +24.2% PF1.04, magic 880002) — 비용 약해 라인업 제외, 필요 시 보관용.

## ▶ MT5 등록 순서 (프리셋 로드 — 수기입력 불필요)

확정 4슬롯을 바로 로드할 `.set` 프리셋을 `mt5/sets/`에 넣어뒀다. 값을 일일이 입력하지 말고 **로드**만 하면 된다.

| 슬롯 | 프리셋 파일 | 차트 종목 | 차트 TF | 매직 |
|------|------------|----------|---------|------|
| ★단기 | `sets/MAPB_US100_M30.set` | US100.x | **M30** | 880001 |
| ★스윙 | `sets/MAPB_US100_H4.set` | US100.x | **H4** | 880004 |
| ★골드스윙 | `sets/MAPB_XAU_H2.set` | XAUUSD | **H2** | 880003 |
| ★골드단기숏 | `sets/MAPB_XAU_M10_SHORT.set` | XAUUSD | **M10** | 880005 |

1. MetaEditor에서 `MA_Pullback_EA.mq5` **F7 컴파일** (한 번만).
2. **`.set` 파일 4개를 MT5 프리셋 폴더로 복사**: 터미널 → `파일 → 데이터 폴더 열기` → `MQL5\Presets\` 안에 넣으면 로드 창에서 바로 보인다. (다른 폴더에 둬도 로드 시 탐색 가능)
3. 슬롯별 **차트를 정확한 종목+TF로 열고**(예: US100.x M30), EA를 차트에 드래그.
4. EA 속성 창 → **입력(Inputs) 탭 → [로드] 버튼 → 해당 `.set` 선택** → 값 자동 채워짐.
5. **InpLot1~N(랏)만 자금에 맞게 조정**. 매직넘버·나머지는 건드리지 말 것.
6. 공통 탭에서 **자동매매 허용** 체크 → 확인. 4개 차트 반복.
7. 터미널 우상단 **[알고리즘 트레이딩] 버튼 ON**.

> ⚠ **반드시 데모/전략테스터 선검증 후 소액 실거래.** US100은 **`.b`가 청산전용**이라 **`US100.x`**로 거래(README 상단 경고 참조). 골드는 헤징계좌·"거래 전체 가능" 재확인.
> ⚠ 4슬롯 모두 **2024-07~2026-06 상승장만** 검증됨 — 하락장 미검증이 최대 리스크. 골드 M10 숏이 하락 전환 시 방어축.
