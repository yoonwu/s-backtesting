# 자동매매 후보 전략 리스트

> 성과 검증을 통과한 전략을 모아두는 곳. "자동매매 후보 리스트 줘"라고 하면 이 표를 보여주면 됨.
> 모든 수치는 **수수료·스프레드 미반영** 백테스트 기준 (2024.07~2026.06, 단 US100 M5는 2025.01~).
> 손익비≥1(진짜 엣지), 파라미터 능선·워크포워드 검증 통과한 것만 등재.

## ✅ 확정 후보

### 1) 골드 H2 (스윙) — `GOLD-H2-LONG`
| 항목 | 값 |
|------|-----|
| 종목 / TF | XAUUSD · H2 (2시간봉) |
| 전략 | 더블비 (기본, 컨펌 X) |
| 방향 | 롱 (매수) |
| 진입 방식 | 단일 (등분1) |
| 손절 (SL) | 1.0 ×TR  *(SL 0.5~1.5 능선 모두 양호)* |
| 익절 (TP) | 3.0 ×TR · 돌파봉 고가 기준(fixed) |
| **성과** | 거래 43 · 승률 67% · 손익비 1.86 · 수익 **+79.8%** · MDD **-3.6%** · **MAR 22.3** |
| 검증 | SL0.5~1.5 × TP2.5~3 구역 전체 MAR 12~22 (넓은 능선) |
| 빈도 | 월 1~2회, 1포지션 수일~수주 보유 (느긋한 스윙) |
| 주의 | 거래 43개로 표본 얇음. 골드 강세장 의존(하락장 미검증) |

## 🔵 대안 후보 (참고)

| 진입 | 단일 (등분1) · 컨펌 윈도우 8 |

| 비고 | 골드 H2와 같은 종목이라 분산효과 낮음. 풀 2년 데이터 + 높은 손익비가 장점 |

## 📌 포트폴리오 메모
- **추천 조합: 골드 H2 + 나스닥 M5** → 종목이 달라 한 계좌 안에서 진짜 분산.
- 한 계좌 다중 전략: **헤징 모드 계좌 + 전략별 매직넘버 분리** 필요. 증거금·리스크는 합산 기준.
- 골드만 묶으면(H2+M12) 상관 높아 분산 효과 적음.

## 변경 이력
- 2026-06-29: 골드 H2, 나스닥 M5 확정 등재 / 골드 M12 대안 등재
